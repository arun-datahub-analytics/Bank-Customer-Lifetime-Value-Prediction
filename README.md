# 🏦 Bank Customer Lifetime Value (CLV) Prediction Using AI

## 📌 Overview

This project focuses on predicting and analyzing Customer Lifetime Value (CLV) in the banking sector using Machine Learning, Deep Learning, Customer Segmentation, and Generative AI.

The objective is to identify high-value customers, predict future customer value, and generate intelligent business recommendations that help improve customer retention and profitability.

---

## 🎯 Project Objectives

✅ Calculate Customer Lifetime Value (CLV)

✅ Segment customers into different value groups

✅ Predict future customer value using Machine Learning

✅ Improve prediction performance using Deep Learning

✅ Generate AI-powered customer retention strategies

✅ Support data-driven banking decisions

---

## 📊 Dataset Information

The dataset contains banking customer information used for customer value analysis and prediction.
